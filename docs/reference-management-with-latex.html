2.1 Install Biber

On Windows: Powershell –> tlmgr install biber (seems to be NOT automatically installed with package biblatex)

2.2 In Texworks

Always run

  1. pdflatex
  2. biber
  3. pdflatex (again)

Or write bash script which runs the comands in this order

2.3 In RStudio

Biber and knitr can make some problems togeter (see here)

Easy workarround: use backend bibtex instead of backend biber

\usepackage[backend=bibtex]{biblatex}
